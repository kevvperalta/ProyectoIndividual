Nombre del proyecto: Explora Perú

url del proyecto hospedado: https://explora-peru.onrender.com

breve descripcion:

Explora Perú es una aplicación web turística que presenta información sobre los principales destinos del país, organizada por departamentos. El sitio incluye cinco páginas informativas (Inicio, Destinos, Recursos, Acerca de, Contacto) y un formulario funcional que permite a los visitantes enviar consultas, las cuales se almacenan en una base de datos MySQL remota. El proyecto implementa un backend en Flask que sirve tanto el contenido estático como la API para procesar los mensajes, estando completamente desplegado en la nube.

Tecnologías Utilizadas:

Frontend: HTML5, CSS, JavaScript
Backend: Python, Flask
Base de Datos: MySQL, SQL
Hospedaje: Render (backend/frontend), Railway (base de datos)

Control de Versiones: Git, GitHub
