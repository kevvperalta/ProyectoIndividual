Sobre este proyecto

Esta aplicación web fue desarrollada como parte de mi proyecto personal del curso de IDWEB. El objetivo es mostrar algunos de los lugares turísticos más representativos del Perú, organizados por departamentos para facilitar su exploración.

Aquí encontrarás información básica, imágenes referenciales y una estructura para navegar por los principales destinos del país.

Sobre el creador

Soy un estudiante de la carrera de ingeniería de sistemas de la UNSA, interesado en el desarrollo web. Este proyecto representa mi primer acercamiento a la creación de una aplicación completa usando HTML, CSS, JavaScript, Python y MySQL.

A medida que avance en mi aprendizaje, planeo mejorar la interfaz, añadir más destinos turísticos y convertir este sitio en una guía turística interactiva del Perú.
